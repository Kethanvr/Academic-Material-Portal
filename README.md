## � Project Overview

> **Live Demo**: [https://ise-study-material.vercel.app](https://www.vtustudymaterials.online/)

The **Academic Material Portal** is a cutting-edge web application designed to revolutionize how students access educational resources. By replacing scattered links with a unified, intuitive, and mobile-responsive interface, this portal ensures that lecture notes, syllabus copies, and question banks are just a click away.

Built with the latest **Next.js 16**, **Tailwind CSS**, and **Shadcn UI**, the platform offers a premium, app-like experience with full **PWA (Progressive Web App)** support and fully dynamic, real-time data fetching.

## ✨ Key Features

### 🎓 For Students
- **Centralized Dashboard**: Instant access to all subject materials and resources.
- **Smart Previews**: Built-in optimized file viewer for PDFs and documents.
- **Instant Downloads**: One-click downloads for offline study.
- **Mobile First**: Fully responsive design with touch-optimized controls.
- **PWA Ready**: Install the app to your home screen and access content anywhere.

### 🛡️ For Administrators
- **Real-Time Dashboard**: Live analytics on subjects, files, and user activity.
- **Dynamic Management**: Drag-and-drop file organization and seamless folder management.
- **Unifed Review System**: Replaces scattered Excel files for **TYT reviews** and consolidates **P, K, & H management reviews** into a single, efficient portal.
- **Audit Logs**: Detailed tracking of all uploads and modifications.

## 🛠️ Tech Stack

This project leverages the latest web technologies to deliver performance and scalability:

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Backend**: Real-time Data Architecture
- **PWA**: Using `next-pwa` with custom Webpack configuration
